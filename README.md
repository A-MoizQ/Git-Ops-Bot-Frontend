# Git-Ops-Bot-Frontend

Automated git ops bot frontend server built with React and Vite.

## Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)

## Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/A-MoizQ/Git-Ops-Bot-Frontend.git
   cd Git-Ops-Bot-Frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:5173` (or another port if 5173 is busy).

## Available Scripts

- `npm run dev` - Start the development server with hot module replacement (HMR)
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build locally

## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **JavaScript** - Programming language

## Development

The development server supports Hot Module Replacement (HMR), so you can see your changes instantly without refreshing the page. Simply edit any file in the `src` directory and save to see the updates.
